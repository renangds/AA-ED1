# AA-ED1
Trabalho para a AA de Estruturas de Dados 1 de 2016.2

<h2>Lista de Contatos</h2>

<p>O trabalho basicamente consiste em criar uma lista de contatos utilizando uma lista encadeada, fazendo operações como: Criar lista, remover contato, criar contato, buscar contato e fazendo operações de leitura em um arquivo, os salvando após as operações de leitura e escrita.</p>
